About

This is a real-time Computational Fluid Dynamics (CFD) simulator using the Lattice Boltzmann Method (LBM). The simulation visualizes fluid flow patterns and demonstrates fundamental fluid dynamics principles.

Features

  • Real-time fluid simulation
  • Interactive viscosity control
  • Adjustable flow speed
  • Visual representation of velocity field
